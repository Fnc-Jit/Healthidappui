## Notes:
- This app uses shadcn/ui components which are built on top of Radix UI primitives
- Tailwind CSS v4.0 is used for styling (no tailwind.config.js needed)
- Sonner is pinned to version 2.0.3 for toast functionality
- All other packages can use their latest versions
